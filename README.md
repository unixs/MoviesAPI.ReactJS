# Movies API Server

Simple API Server and scraper for themoviedb

Find more info in `api.yml`

## Get started ##

### Install all dependencies

```bash
npm install
```

### Run unit tests

```bash
npm run test
```

### Lint your code

```bash
npm run lint
```
